## Experiment Chain: TASK-9 → TASK-10 → TASK-11

### Root cause (established before TASK-9)
Within-sector IC collapses to near-zero in India bull markets (2016-2019).
Momentum model assigns spread scores in trending markets, but the spread is trend-driven not signal-driven.
Fixed `top_k=5` concentrates noise in bull regimes. Horizon shift (TASK-8) can't fix this.

### TASK-9 — Adaptive top-k via dispersion percentile [CLOSED → led to TASK-11]
**Design**: p90-p10 spread of rank scores, percentile-ranked vs trailing 8 rebalances.
- `pctile < 0.33` → k=7 (weak signal, diversify)
- `0.33–0.67` → k=5 (default)
- `pctile > 0.67` → k=3 (strong signal, concentrate)

**Results (2013-2020)**:
- `within_sector_ic`: +0.032 (vs -0.021 baseline) ✅
- Per-year: 2015=+0.333, 2016=**-0.121**, 2017=+0.113, 2018=-0.048, 2019=-0.042, 2020=+0.127
- CAGR: 12.39% (vs 10.64% baseline) ✅
- **2016 failure**: high trend → low within-sector dispersion → adaptive chose k=7 → held reversal stocks → IC = -0.121

**Structural finding**: dispersion ≠ trend signal. In bull regimes, low dispersion triggers k=7 (correct for IC) but k=7 holds more reversal stocks.

### TASK-10 — Single condition trend gate at p70 [REJECTED]
**Design**: if `abs(nifty_3m_pctile) > 0.70` → k=mid_k=5

**Results**:
- Overall IC: +0.026 (regression from +0.032) ❌
- 2016: -0.088 (improved) but 2017: +0.032 (degraded from +0.113) ❌
- CAGR: 10.93% (reject zone) ❌
- **Root cause**: gate fired in 2017/2020 where adaptive was working correctly. Blunt macro condition with no stock-level check.

### TASK-11 — Combined gate: trend AND weak dispersion [PROMOTED TO FULL RL]
**Design**: gate fires only when BOTH:
1. `abs(nifty_3m_abs_pctile) > 0.80` (strong macro trend, trailing 12 rebalances)
2. `dispersion_pctile < 0.33` (within-sector scores are weak/clustered, bottom third of history)

When gate fires: `k = mid_k = 5`, dispersion history NOT updated (frozen).
When gate doesn't fire despite trend: adaptive runs normally (2017/2020 case — trend but real signal).

**Results (2013-2020)**:
- `within_sector_ic`: **+0.045** (best ever, +0.013 over TASK-9) ✅
- Per-year: 2015=+0.333, 2016=-0.090, 2017=**+0.121**, 2018=-0.020, 2019=-0.036, 2020=+0.124
- CAGR: 11.02% (below 12.0% gate — see tradeoff note) ⚠️
- 2016 gate: -0.090, still below -0.021 success → confirmed as ranker limitation not gating bug

**CAGR tradeoff**: gate forces k=5 instead of k=7 in bull+weak-IC periods. k=7 passively captures more trend upside even with near-zero IC. RL layer expected to compensate by increasing aggressiveness/exposure in these periods.

**Decision**: Promote to full RL — best IC ever, 2017/2020 structural bugs fixed, 2016 correctly diagnosed.
